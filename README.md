# E-Commerce Web Application

## Overview
The E-Commerce Web Application is a full-stack web project designed to provide an online shopping experience. 
Users can browse products, add items to their cart, and place orders. The application includes role-based access control for Admin and User accounts, backend APIs for product and order management, and MySQL database integration for data storage.

## Features
- Product Catalog Management
- Add to Cart Functionality
- Checkout and Order Placement
- User Login and Authentication
- Role-Based Access Control (Admin/User)
- RESTful APIs for Product and Order Management
- MySQL Database Integration
- Responsive and User-Friendly Interface

## Expected Outcome
- Display available products.
- Allow users to add products to the cart.
- Process orders through checkout.
- Store product and order information in MySQL.
- Support role-based access for Admin and User accounts.
